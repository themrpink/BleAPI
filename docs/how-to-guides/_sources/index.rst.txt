
Welcome to CosmedBleLib's documentation
========================================

The CosmedBleLib Library offers client side communication with the Bluetooth Low Energy protocol stack in Windows 10.
The library is built on the Bluetooth Low Energy UWP API (see https://docs.microsoft.com/en-us/windows/uwp/devices-sensors/bluetooth-low-energy-overview for Microsoft documentation).
It's written in C# for .NET Framework 4.8.
